Laissez votre travail parler pour lui-même.

Il ne devrait pas importe où vous êtes-les États-Unis, l'Ukraine, une île, en Inde ou ailleurs. Si votre code résout un problème, vous devriez être payé. La qualité de votre travail devrait être tout ce qui compte lors de la fourniture d'une solution à un problème. Avec l'argent git, votre travail est votre curriculum vitae. Rien d'autre ne compte.

Pas un codeur? Pas de problème.

Un problème n'a pas besoin d'être technique. Les gens ont des besoins pour les graphiques, le texte, le montage vidéo, l'aide des médias sociaux et bien d'autres choses - et ils sont prêts à payer pour cela. Si vous avez une compétence , il pourrait y avoir de l'argent qui vous attend.

Comment cela fonctionne.

Trouvez un projet de Bitcoin-payer et profiter du commerce sans frontières.

opportunités git

Parcourir les numéros de Bitcoin-payable actuelles sur GitHub. Toutes les questions avec un paiement contiendra l'étiquette git argent vert avec un "Bounty actuel:" lorsqu'on regarde. Le paiement en cours pour résoudre la question est notée dans la CTB ( Bitcoin ) et en dollars US . Si vous avez une question à propos de la question, ne hésitez pas à commenter et poser! Nouveau à git?

travail git

Une fois que vous avez trouvé une occasion, vous êtes à la maison à mi-chemin (ou partout où il est vous allez!). Allez-y et se rendre au travail! Tout ce que vous devez faire est de cloner le repo, créer une branche, vos changements et soumettre une demande de traction. Dans votre message de demande de traction, assurez-vous de noter qu'il résout le problème et inclure également votre adresse Bitcoin. Nouveau sur Bitcoin?

argent git

Une fois votre demande de traction est fusionné, vous êtes payé instantanément. Cependant, vous choisissez à git l'argent est à vous. Certaines personnes recherchent des petits problèmes qui ajoutent à des sommes plus importantes. D'autres vont directement pour les gros travaux. Peut-être que vous êtes milieu de la route, la cueillette et le choix de faire de l'argent supplémentaire sur le côté. Il est complètement à vous comment vous voulez git l'argent.
