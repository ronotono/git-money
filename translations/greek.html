Οι άνθρωποι έχουν προβλήματα που χρειάζονται διόρθωση. Εσύ έχεις την δυνατότητα να τα λύσεις. Αυτή είναι η ευκαιρία σου.

  • Ελέυθερη αγορά
  • Αυτόματοποιημένες πληρωμές
  • Πολύ λίγοι φόροι
  • Εργασία απο παντού

Αφήστε την δουλειά σας να μιλήσει για εσάς.

Δεν πρέπει να έχει σημασία πού βρίσκεστε - στις Ηνωμένες Πολιτείες, στην Ουκρανία, σε ένα νησί, στην Ινδία ή οπουδήποτε. Εάν ο κωδικάς σας λύνει ένα πρόβλημα, θα πρέπει να πληρωθεί. Η ποιότητα της εργασίας σας θα πρέπει να είναι αυτό που έχει σημασία κατά την παροχή μιας λύσης σε ένα πρόβλημα. Με το git money, η δουλειά σας είναι το βιογραφικό σας. Τίποτα άλλο δεν παίζει ρόλο.

Δεν είστε προγραμματιστής; Δεν υπάρχει πρόβλημα.

Ένα πρόβλημα δεν πρέπει να είναι υποχρεωτικά τεχνικό. Οι άνθρωποι έχουν ανάγκες για γραφικά, κείμενο, μοντάζ βίντεο, βοήθεια στα μέσα κοινωνικής δικτύωσης και πολλά άλλα πράγματα - και είναι πρόθυμοι να πληρώσουν για αυτό. Εάν έχετε μια ικανότητα,μπορεί να υπάρχουν χρήματα που σας περιμένουν.

Βρείτε ένα έργο πληρωτέο σε bitcoin - και απολαύστε χωρίς σύνορα εμπόριο.

Ευκαιρίες στο git

Δείτε τα υπάρχοντα έργα που πληρώνουν με bitcoin στο GitHub. Όλα τα θέματα με πληρωμή θα περιέχουν την πράσινη ετικέτα του git money μαζί με την"Υπάρχουσα Αμοιβή:" όταν τα βλέπετε. Η πληρωμή για να λύσετε το θέμα είναι σημειωμένο σε BTC (Bitcoin) και σε Δολλάρια Αμερικής. Αν έχετε ερωτήματα σχετικά με το θέμα, απλά ρωτήστε σχολιάζοντας! Είστε νέος στο git?

Εργασία στο git

Μόλις έχετε βρει την ευκαιρία, είστε στα μισά του δρόμου για το σπίτι (ή όπου εσείς πηγαίνετε!). Προχωρήστε με την δουλειά! Το μόνο που χρειάζεται να κάνετε είναι ένας κλώνος του κώδικα, δημιουργήστε ένα παρακλάδι, εφαρμόστε τις αλλαγές και δημιουργήστε ένα αίτημα. Στο μήνυμα της αίτησης σας, βεβαιωθείτε οτι λύνει το πρόβλημα και συμπεριλάβετε επίσης τη διεύθυνση bitcoin σας. Είστε νέος στο Bitcoin;

git money

Μόλις το αίτημά σας συγχωνευθεί, μπορείτε αμέσως να πληρωθείτε. Ωστόσο, το αν θα επιλέξετε το git money εξαρτάται απο εσάς. Μερικοί άνθρωποι ψάχνουν για μικρά ζητήματα που προσθέτουν σε ένα μεγαλύτερο. Άλλοι πάνε κατ 'ευθείαν για τις μεγάλες εργασίες. Ίσως είστε στη μέση του δρόμου, και επιλέξετε να βγάλετε μερικά παραπάνω χρήματα. Εξαρτάτε από εσάς και το πως θέλετε να χρησιμοποιήσετε το git money.
